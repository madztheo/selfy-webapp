# Selfy - Frontend

This repos contain everything to interact with the Selfy smart contract.



## Getting Started

First install the package :

```bash
npm install
# or
yarn install
# or
pnpm install
```

And then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
